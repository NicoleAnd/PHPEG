4 前台语言和后台语言的区别

    PHP: Hypertext PreProcessor 超文本预处理器

    我们在浏览器中输入 http://localhost/01.php发生了什么？
    S1 HTTP request 请求  
    S2 php开始工作 <?php  ?>语句在服务器上执行完毕，把页面压平为html传输给用户
    S3 HTTP response响应
    S4 浏览器开始执行js、html、css等代码

    * JS、CSS、html在客户端、浏览器端执行 叫前台脚本
      php、jsp、asp都是在服务器端运行叫后台语言。任何后台语言，都要部署环境。 

    * js运行在客户端、php运行在服务器
      所有的数据增删改查都是服务器的业务 都是php的业务
      所有页面的样式、运动、交互、事件、表单验证都是js业务

5 php基本语法
  *书写位置 
        <?php   ?>不能嵌套 出现在任何位置

        *.html  会被视为静态页面  不解析
        *.php 

   * 输出语句 
       php要求所有的语句必须加分号

       echo "something"; // 输出的是页面当前位置  
       或者 <?php  echo(2+6*4); ?>  

       echo默认不换行 假如想换行 自己用.拼接标签

   * 变量
      变量以$开头,后面是名字
      名字不能以数字开头，必须是下划线或字母开头
      名字可以有数字、下划线、字母
      区分大小写 比如$a  $A 不同的变量


      php变量也是弱类型，不需要定义什么类型 传什么类型就是什么类型


    * php作用域
        php也没有块级作用域
         {

         }  


         一个全局变量，在函数内部是不认识的，若想让函数内部认识全局变量，需要在函数内部书写global关键字            

    * php运算符
         基本与js一样，就是连字符，php连字符是.


    * 数组
         ***
          print_r($a); 打印数组的全貌
          count($a); 数组长度

    * 语句
       和js一样
       if   if.....else    for  while  do-while switch

    * 函数
        js中函数有声明头提升，php必须先定义函数 才能调用
        函数调用()一种调用方式
        函数传参和返回值与js一样
